# Tower Defense 101

Your first tower defense challenge. Build and upgrade towers to stop enemy waves.

## Features
- Multiple tower types
- Wave-based gameplay
- Upgrade system
- Strategy planning
- Infinite replayability

## Build
```bash
npm install
npx cap sync
npx cap open android
npx cap open ios
```

## Tech Stack
- React + Vite
- Capacitor
- iOS & Android
